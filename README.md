## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```
- node

### Setup

Install the dependencies and devdependencies.

```sh
  cd my-portfolio

  npm install

```

To start react server

```sh

npm run dev

```
